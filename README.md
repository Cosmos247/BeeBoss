# BeeBoss

Telegram userbot, який збирає рейтинги гри "Тааємниці Королівства Остромаг": загальний рейтинг гільдій і рейтинг учасників двох гільдій (КАБ і ФАБ) з двох Telegram-аккаунтів. В заданий час знімає бейзлайн, а на команду — порівнює і присилає звіт.

## Команди

- `/diff` — приріст слави гільдій з останнього бейзлайну
- `/chlen` — приріст слави гільдійців **КАБ**
- `/chlenfab` — приріст слави гільдійців **ФАБ** (дані з другого аккаунта)
- `/settime HH:MM` — змінити час щоденного бейзлайну (тільки твої повідомлення)

**Авто-звіт у визначений чат щодня** (тільки твої повідомлення):
- `/setreportchat` — встановити поточний чат як цільовий для авто-звіту
- `/clearreportchat` — вимкнути авто-звіт
- `/setreporttime HH:MM` — змінити час авто-звіту (default `11:55`)
- `/autoreport` — показати поточні налаштування

При увімкненому авто-звіті бот сам викликає `/diff`, `/chlen`, `/chlenfab` у заданий час. Якщо аккаунт у бою — повторна спроба через 90с. Якщо знов в бою — у чат прийде повідомлення `⚔️ Зараз я в бою...`.

Кулдаун `/diff`, `/chlen`, `/chlenfab` — 5 хвилин на кожну команду окремо.

## Установка

Потрібен **Python 3.12** (Pyrogram несумісний з 3.14).

```bash
git clone <url> beeboss && cd beeboss
python3.12 -m venv .venv
.venv/bin/pip install -r requirements.txt
cp .env.example .env
# відкрий .env і встав свої api_id/api_hash з https://my.telegram.org
.venv/bin/python main.py
```

При першому запуску Pyrogram **двічі** запитає номер телефону + код з Telegram — спочатку для КАБ-аккаунта (створиться `my_account.session`), потім для ФАБ-аккаунта (створиться `fab_account.session`). Наступні запуски без авторизації.
